<b>STRING ROTATION</b>

<b>CHALLENGE DESCRIPTION:</b>
    You are given two strings. Determine if the second string is a rotation of the first string.

<b>INPUT SAMPLE:</b>
	Your program should accept as its first argument a path to a filename. Each line in this file contains two comma separated strings. 

    E.g.
    Hello,lloHe
    Basefont,tBasefon

<b>OUTPUT SAMPLE:</b>
    Print out True/False if the second string is a rotation of the first.

    E.g.
    True
    True
